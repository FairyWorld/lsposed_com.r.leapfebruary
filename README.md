# LeapFebruary(闰二月)

一个小而美的微信模块，功能陆续更新中

## [常见问题点我](./FAQ.md)  

## 使用说明  

+ 目前支持的微信版本：  
官网版本:8.0.45、8.0.46、8.0.47、8.0.48、8.0.49（首推荐版本）  
GooglePlay 版本:8.0.42（2424）、8.0.42（2429）
+ 模块激活后(如果你不知道什么是模块激活，先看一下[常见问题](./FAQ.md) )，在微信内点击“我->设置->插件”可进入模块设置界面
